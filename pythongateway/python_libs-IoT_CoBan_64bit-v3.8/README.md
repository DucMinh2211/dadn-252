# IoT Gateway - Hệ thống Nhà Thông Minh
Dự án Gateway trung gian kết nối phần cứng Yolo:Bit với máy chủ Adafruit IO.

## 1. Yêu cầu hệ thống (Prerequisites)
* Ngôn ngữ: Python 3.x
* Hệ điều hành: Windows 64-bit

## 2. Cài đặt thư viện (Offline Installation)
1. Giải nén và copy toàn bộ thư mục `python_libs-3.x` vào cùng cấp với file `main.py`.
2. Mở Terminal (hoặc Command Prompt) tại thư mục chứa dự án.
3. Chạy lệnh sau để cài đặt:
   ```bash
   python setup.py
